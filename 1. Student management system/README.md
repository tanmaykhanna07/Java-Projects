This is a console-based Student Management System built using Core Java to practice Object-Oriented Programming concepts. The application allows users to add and view student records through a menu-driven interface. It demonstrates proper use of classes, objects, constructors, constructor overloading, dynamic data storage using ArrayList, and clear separation of concerns by keeping user interaction in the main class and business logic in a dedicated StudentManager class. The project focuses on clean program structure, correct object modeling, and real-world Java design practices rather than UI.
