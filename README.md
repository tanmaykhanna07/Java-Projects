# Java Projects

This repository contains small Java projects built to practice object-oriented programming and backend fundamentals.

## Projects

### Student Management System
A basic system that supports student record management using core OOP principles such as encapsulation and abstraction.

### Merchant Payment Gateway
A simulation of different payment methods implemented using inheritance and polymorphism.

### Library Management System
A structured system with different roles and item types. Uses enums for status handling and demonstrates clean object modeling.

### Notification System
A basic system that uses interfaces to push different types of notifications to the user.
## Technologies Used

- Java
- Object-Oriented Programming
- IntelliJ IDEA

## Purpose

The goal of these projects is to strengthen understanding of:

- Class design
- Inheritance and polymorphism
- Separation of responsibilities
- Clean and maintainable code structure
